# Магазин – что за зверь и как запустить

Простая штука для учёта продаж. Товары, корзина, кассиры, отчёты по дням. На фласке.

## Что нужно сделать

1. `git clone https://github.com/platonzhuman/informatice_project/tree/main/shop`

2. Установить зависимости (только Flask):
   ```bash
   pip install -r requirements.txt
   ```
   или просто `pip install flask`

3. Запустить сервер:
   ```bash
   python app.py
   ```

4. Открыть браузер и перейти по адресу:
   ```
   http://127.0.0.1:5000
   ```

## Как пользоваться

да как обычным магазином :)

## Если что-то сломалось

Удали файл `shop.db` и перезапусти – создастся заново, ведь это всегда сработает :_:
